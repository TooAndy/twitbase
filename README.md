# HBase In Action: TwitBase

## 初衷
<HBase实战>中的例子是根据HBase 0.92开发的, 和现在的1.x版本有很多不同.
因此学起来会遇到很多想象不到的困难.
因此, 结合官方文档和原书代码, 对书内api进行了一些更新,希望能帮助大家学习.

## HBase版本
该项目API使用1.2.0-cdh5.7.0版本
